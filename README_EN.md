## ğŸ› ï¸ How to Contribute?

Gelisim Nexus operates on a "Wiki" principle. If you possess course notes or have insights into international programs, you can contribute to the community:

1. **Adding Notes:** Navigate to the relevant faculty repository and create documentation for missing courses.
2. **Peer Review:** If you spot an error in existing notes, use the "Edit" button to suggest a fix via Pull Request.
3. **Mentorship:** Share your personal experiences regarding ESC, Erasmus, or NGO projects in the `Global Opportunities` section.
